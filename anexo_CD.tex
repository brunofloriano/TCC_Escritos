\chapter{Descrição do conteúdo do CD}\label{AnCD}
%O CD que acompanha este trabalho contém os códigos trabalhados na pasta \emph{quad}. O arquivo \textit{main.cpp} é o principal, contendo as funções de inicialização e a função que define a thread de controle. O arquio \textit{command.cpp} abrange os comandos de escrita e leitura dos motores. Os arquivos \textit{gdatalogger.c}, \textit{gmatlabdatafile.c} e \textit{gqueue.c}


\begin{enumerate}
	\item Programas em C++ para inicializações, definição da thread de controle, filtragem, medição e comandos. A execução do arquivo "main" \space executará estas funções na ordem correta.
	\item Programa em C para implementação do gDataLogger
	\item Diretório com os arquivos do Matlab ".mat" \space obtidos para a aquisição de dados
	\item Arquivos do Matlab ".m" \space para a análise dos dados. O arquivo "analise\_vmotores\_new.m" \space analisa os dados dos sistemas sem paralelismo (tanto do modo posição quando do modo velocidade). O arquivo "analise\_pmotores.m" \space trata os dados do sistema com paralelismo
\end{enumerate}
